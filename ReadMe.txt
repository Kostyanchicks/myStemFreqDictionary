Как пользоваться программой:
1) Распаковать архив в любом месте;
2) Загрузить нужный текст в директорию Частотный словарь\bin\Debug под названием text.txt;
ВАЖНО! Файл text.txt может иметь любое содержание на русском языке,
но при условии, что файл сохранён в кодировке UTF-8.
3) Запустить исполнительный файл Частотный словарь.exe.
